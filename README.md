# Tic-Tac-Toe Game - Java (GUI Application)
A beginner friendly Tic-Tac-Toe game built using Java Swing.
This game allows two players to play on the same computer with a simple and interactive graphical interface.

# > Features:  
  - Two-player gameplay(X and O)
  - Graphical User Interface using Swing
  - Button based input for moves
  - Real-time update of the game board
  - Handles invalid moves (cannot overwrite a cell)
  - Declares winner or draw automatically

# > Technologies Used:  
  - Java SE
  - Java Swing
  - Object-Oriented Programming(OOP) Principles

# > How to run:  
1. Make sure you have Java installed on your machine
   - `java --version`
3. Compile the program
   - `javac TicTacToe.java`
4. Run the program
   - `java TicTacToe`
